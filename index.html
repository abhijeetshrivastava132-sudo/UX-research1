GSAP ScrollTrigger

Example implementation for split-text animation:

gsap.from(".split-word", {
    y: 50,
    opacity: 0,
    stagger: 0.03,
    scrollTrigger: {
        trigger: ".section",
        start: "top 80%",
        toggleActions: "play none none reverse"
    }
});

This creates a word-by-word reveal as the user scrolls into view.

07

Identified Problems

  • Scroll fatigue — 35% of users reported dizziness with excessive staggered animations
  • CPU throttling — Split-text re-triggers on every scroll on low-end devices
  • Hover unusable on mobile — No tap feedback alternative provided
  • No reduce motion toggle — Violates WCAG accessibility guidelines
08

Conclusion & Recommendations

Recommended Approach

Split-text only for hero and section titles
Add prefers-reduced-motion media query
Limit hover transitions to 200-300ms

Avoid These Patterns

Staggering every paragraph element
Heavy simultaneous scroll and hover
Relying only on hover for CTAs

Usability score: 7.8 out of 10

Animations boost brand recall but must respect performance budgets.

Next steps:
1. A/B test reduced motion mode
2. Optimize ScrollTrigger for mobile devices
